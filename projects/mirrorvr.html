MirrorVR

A powerful VR wrapper for Mirror Networking.

What is MirrorVR?

MirrorVR is a wrapper for Mirror Networking that adds easy VR multiplayer support. It simplifies networking VR rigs, login systems, and cross-platform connectivity.

Why MirrorVR?

Feature MirrorVR PhotonVR
Cost 100% free ✅ 20 CCU free, after that very expensive ❌
Cosmetics Server-side validation before equipping ✅ Insecure and fully client sided ❌
Login Uses EOS, Oculus validation, rate-limiting ✅ Easily mimicked and spammed ❌
Efficiency Fast 🔥 Slow networking ❌
Bandwidth Unlimited ✅ 60GB free / month ❌
CCU 64 players per lobby, unlimited lobbies ✅ 20 CCU free ❌

Supported Devices

  • Windows 10/11 (64-bit & ARM)
  • MacOS Intel & ARM
  • Linux (x64 & ARM)
  • Meta Quest 2 / Pro / 3 / 3S

Supported Login Methods

  • Oculus Login (User ID + Nonce)
  • Epic Games Account (EOS Auth)
  • Steam Session Ticket
  • Device / Hardware ID fallback

Features

  • Oculus Authentication
  • Cross-Platform Multiplayer
  • Usernames
  • Join Lobbies via Name
  • Banning / Sanctions
  • Player Colors
  • Cosmetics System
  • Steam Login Support
  • Host Migration (WIP)
